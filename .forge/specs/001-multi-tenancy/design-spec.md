#### Responsive Notes

- **1440px**: Full layout as wireframed. Sidebar expanded (240px). 4-column stat cards. Table with all columns.
- **1024px**: Sidebar collapsed to icons (64px). Stat cards stack 5 across (compact). Actions column becomes overflow menu (â‹®).
- **768px**: Sidebar hidden (hamburger toggle). Stat cards stack 3+2. Table becomes card layout. Actions as full-width buttons within each card.
