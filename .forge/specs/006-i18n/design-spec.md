## 1. Overview

| Field         | Value                                                                            |
| ------------- | -------------------------------------------------------------------------------- |
| Feature       | Internationalization (i18n) — namespace-based translations with tenant overrides |
| Spec ID       | 006-i18n                                                                         |
| Platform      | Web SPA (React 19 + Vite + TanStack Router)                                      |
| Viewports     | Desktop (1440px), Tablet (1024px → 768px), Mobile (375px)                        |
| Design system | Tailwind CSS + `@plexica/ui` component library (v1.2)                            |

**Design approach**: Desktop-first. The Override Editor is an admin tool
favoring information density. The Language Selector is high-visibility and
must work well at all breakpoints.

#### Responsive Notes — Language Selector

- **1440px**: Globe icon in header bar alongside notification bell and avatar. Dropdown opens below with 200px width
- **1024px**: Same layout, header items may collapse — Globe icon remains visible
- **768px**: Globe icon moves to a compact icon-only header. Dropdown renders as bottom sheet on tablet
- **375px**: Globe icon is a single icon in the header. Dropdown opens as a full-width bottom sheet for easier tap targets

#### State Inventory — Override Editor

| State                | Trigger                                         | Visual Change                                                                                                         | User Feedback                           |
| -------------------- | ----------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- | --------------------------------------- |
| Default              | Page load                                       | Table loads with all translation keys for selected locale, paginated                                                  | —                                       |
| Loading              | Initial load or locale change                   | `Skeleton` rows in table (6 rows)                                                                                     | "Loading translations..." (aria-live)   |
| Error (API)          | `GET /tenant/translations/overrides` fails      | `Alert` banner: "Unable to load overrides. Please try again." + Retry button                                          | Error alert with retry CTA              |
| Empty (no keys)      | No enabled plugins, or no keys match filter     | `EmptyState`: "No translation keys found." CTA: "Clear filters"                                                       | —                                       |
| Empty (no overrides) | Keys exist but no overrides yet                 | Table shows all keys with empty Override column. Info banner: "No overrides configured. Click a key to customize it." | —                                       |
| Dirty (unsaved)      | User has edited at least one override           | Bottom bar appears: "Unsaved changes: N" + [Discard] + [Save]                                                         | —                                       |
| Saving               | Save button clicked                             | Save button shows `Spinner` + "Saving..."                                                                             | aria-busy="true" on save button         |
| Success              | Save completes                                  | Success `Toast`: "Translation overrides saved." Dirty indicator clears                                                | Toast announced via aria-live           |
| Save Error           | `PUT` returns 4xx/5xx                           | Error `Toast` with specific message (e.g., payload too large, validation error)                                       | Toast with actionable error             |
| Orphaned key         | Override references removed plugin key (FR-014) | Row shows `AlertTriangle` icon + "Orphaned" `Badge` (variant: warning)                                                | Tooltip: "This key no longer exists..." |

| Element                 | Type                                    | Action | Outcome                                                                | FR Ref         |

#### Responsive Notes — Override Editor

- **1440px**: Full 3-column table (Key, Original, Override) with sidebar navigation visible. Inline editor expands within the table
- **1024px**: Sidebar collapses to icons-only. Table retains 3 columns but with tighter padding
- **768px**: Sidebar hidden (hamburger). Table stacks to 2 columns: Key + Original on row 1, Override on row 2
- **375px**: Sidebar off-screen. Keys displayed as stacked cards (key name, original, override vertically). Bottom save bar is sticky

### Screen 3: Translation Preview Panel

**Source**: FR-007, FR-013 (spec §9: "changes visible immediately in a preview panel before saving") | **Type**: New

The Translation Preview Panel is embedded within the Override Editor's expanded
row view. It provides a live rendering of the override, including ICU
MessageFormat interpolation.

```
+----------------------------------------------------------------------+
| ▼ crm:deals.count                                   [Namespace: CRM] |
|                                                                       |
|   Original (CRM plugin, en):                                         |
|   ┌───────────────────────────────────────────────────────┐           |
|   │ {count, plural, =0 {No deals} one {# deal}           │           |
|   │                 other {# deals}}                      │           |
|   └───────────────────────────────────────────────────────┘           |
|                                                                       |
|   Override for tenant:                                                |
|   ┌───────────────────────────────────────────────────────┐           |
|   │ {count, plural, =0 {No opportunities} one {# opportu │           |
|   │                 nity} other {# opportunities}}        │           |
|   └───────────────────────────────────────────────────────┘           |
|                                                                       |
|   ┌─── Preview ──────────────────────────────────────────┐    FR-013 |
|   │                                                      │           |
|   │  Test values:                                        │           |
|   │  count = [▾ 0  ]                                     │           |
|   │                                                      │           |
|   │  ┌──────────────────────────────────────────────┐    │           |
|   │  │ Rendered output:                             │    │           |
|   │  │                                              │    │           |
|   │  │   Original: "No deals"                       │    │           |
|   │  │   Override:  "No opportunities"              │    │           |
|   │  │                                              │    │           |
|   │  │  count=0:  "No opportunities"                │    │           |
|   │  │  count=1:  "1 opportunity"                   │    │           |
|   │  │  count=5:  "5 opportunities"                 │    │           |
|   │  └──────────────────────────────────────────────┘    │           |
|   │                                                      │           |
|   │  ⓘ ICU MessageFormat syntax.                         │           |
|   │    Uses {variable} placeholders and plural rules.    │           |
|   └──────────────────────────────────────────────────────┘           |
|                                                                       |
|                            [ Clear Override ]  [ Apply ]              |
+----------------------------------------------------------------------+
```

#### State Inventory — Preview Panel

| State                | Trigger                                      | Visual Change                                                                                                        | User Feedback              |
| -------------------- | -------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | -------------------------- |
| Default              | Row expanded, override input empty           | Preview area shows: "Enter an override to see a preview."                                                            | —                          |
| Live preview         | User types in override input                 | Preview renders the ICU MessageFormat output in real-time. For plurals, shows all categories (zero, one, other)      | —                          |
| ICU syntax error     | User enters malformed ICU MessageFormat      | Preview area shows red-bordered error: "Invalid ICU MessageFormat syntax. Check your plural rules and curly braces." | Inline error below preview |
| Simple text (no ICU) | Override is plain text without `{variables}` | Preview shows single rendered line without test value inputs                                                         | —                          |
| With variables       | Override contains `{variable}` placeholders  | Test value inputs appear for each detected variable                                                                  | —                          |

#### Interactive Elements — Preview Panel

| Element                          | Type                      | Action      | Outcome                                                                           | FR Ref |
| -------------------------------- | ------------------------- | ----------- | --------------------------------------------------------------------------------- | ------ |
| Test value input (e.g., `count`) | `Input` (number/text, sm) | type/change | Updates rendered preview output in real-time                                      | FR-013 |
| Rendered output                  | read-only text block      | —           | Shows formatted translation for each plural category                              | FR-013 |
| ICU info link                    | `Tooltip` on info icon    | hover/focus | Shows: "ICU MessageFormat syntax. Uses {variable} placeholders and plural rules." | FR-013 |

#### Accessibility — Preview Panel

```

Contrast:
  Preview text on --muted: --foreground on --muted = 7.1:1 ✓
  Error text: --destructive on --card = 5.8:1 ✓
  Info text: --muted-foreground on --card = 4.7:1 ✓

  Tab: cycles through test value inputs and action buttons

Screen reader flow:
  1. "Translation preview for crm:deals.count"

  3. "Rendered output: Original: No deals. Override: No opportunities."
```
