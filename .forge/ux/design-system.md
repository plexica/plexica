## 8. Animation / Motion

| Token               | Value                          | Usage                                 |
| ------------------- | ------------------------------ | ------------------------------------- |
| `--duration-fast`   | `100ms`                        | Button state changes, toggles         |
| `--duration-normal` | `200ms`                        | Dropdowns, tooltips, sidebar collapse |
| `--duration-slow`   | `300ms`                        | Page transitions, modal open/close    |
| `--easing-default`  | `cubic-bezier(0.4, 0, 0.2, 1)` | General transitions                   |
| `--easing-in`       | `cubic-bezier(0.4, 0, 1, 1)`   | Element entering                      |
| `--easing-out`      | `cubic-bezier(0, 0, 0.2, 1)`   | Element exiting                       |

**Motion principles:**

- Respect `prefers-reduced-motion`: disable all non-essential animation
- No animation duration > 300ms
- No auto-playing animation that cannot be paused
